Description :
-------------

 Générateur de code C# ou Vb à partir d'un schéma XSD.
 Ce générateur fabrique des classes de données pour WPF grace à l'implémentation d'INotifyPropertyChanged.

Usage :
-------
 Xsd2Gen.exe xsdfile namespace outputfile [cs|vb]
